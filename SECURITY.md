# 🛡️ NexusCore Security Policy

## Supported Versions
Only active branches currently hosted on our production endpoints or active primary repositories receive consistent security patches.

| Version | Supported |
| ------- | --------- |
| Main Dev | LATEST ✅  |
| Archives | Unsupported ❌ |

## 🚨 Reporting a Vulnerability
If you discover a security flaw or an exposed token within any NexusCore repository, **do not open a public issue.** 

Please handle vulnerabilities responsibly:
1. Email the core administrative leads directly at your private point of contact (e.g., your primary email or core discord).
2. Detail the exploit vectors and steps to replicate.
3. Our management team will coordinate an internal patch review before deployment.
